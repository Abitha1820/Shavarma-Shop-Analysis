# Fast Food Sales Analysis

## 📊 Project Overview
This project presents a Power BI dashboard analyzing fast food (shawarma shop) sales data. It provides insights into sales performance, customer preferences, and business trends to support data-driven decision-making.

## 🔧 Tools Used
- Power BI  
- Excel  

## 📈 Key Insights
- Total Sales: ₹32.6L+ and Quantity Sold: 26K+  
- Identified peak sales hour as 1 PM  
- BBQ Shawarma is the top-selling item  
- Analyzed weekly and monthly sales trends  
- Studied hourly sales distribution to identify busy periods  
- Evaluated payment methods (Cash, UPI, Card)  
- Compared performance of different menu items
